Config file for kitty,sway,waybar,and wofi


------Script-----


Updates system, enables multilib, installs pacman packages (sway, waybar, wofi, kitty, steam, discord, nautilus, fastfetch, flatpak, git, base-devel, Font Awesome, Nerd Fonts), installs MissionCenter and Laser via Flatpak, installs yay, installs Helium Browser, ProtonUp, and VS Code from the AUR, runs ProtonUp to install GE-Proton, clones the config repo and deploys all config files to ~/.config/, and adds fastfetch to ~/.bashrc.
