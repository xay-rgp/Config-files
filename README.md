Config file for kitty,sway,waybar,and wofi


------Script-----


Updates system, installs pacman packages (sway, waybar, wofi, kitty, steam, discord, nautilus, flatpak, git, base-devel), installs MissionCenter and Laser via Flatpak, installs yay, installs Helium Browser, ProtonUp, and VS Code from the AUR, then clones the config repo and deploys all config files to ~/.config/.
